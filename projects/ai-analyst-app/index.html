AI Analyst App

A multi-agent financial analysis application that separates CFO, Analyst, Trader, and Fund Manager roles into a structured analysis workflow.

Problem Context

Financial analysis applications need role-specific reasoning, repeatable report generation, and a way to preserve previous analysis results for review.

What I Did

  • Designed a multi-agent workflow with role-specific agents such as CFO, Analyst, Trader, and Fund Manager.
  • Implemented service architecture using FastAPI and Streamlit.
  • Built persistence for analysis history and report lookup.
  • Packaged the application with Docker Compose to make demos reproducible.

Outcome and Portfolio Value

  • Converted an AI analysis concept into a runnable service structure.
  • Practiced separating user interface, backend API, workflow logic, and storage responsibilities.
  • Created a portfolio case for AI service implementation, not only prompt design.
